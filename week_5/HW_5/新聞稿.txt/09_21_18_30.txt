
				Amazon 在 9 月 20 日打出一波機海戰術，推出 11 款智慧家居產品。從新款的 Echo 智慧音箱到全新的智慧微波爐，Amazon 已經下定決心要攻佔家庭的每一個角落。 Amazon 發表的新產品當中有 4 款是新版本的 Echo 智慧音箱，分別是 Echo Dot、Echo Auto、Echo Show 和 Echo Plus。有 5 款產品則是連結 Echo 智慧音箱的智慧家具，包括 Echo Wall Clock、Echo Input、Echo Sub、Echo Link 和 Echo Link Amp。另外還有兩款智慧家居產品，分別是 Amazon 智慧微波爐（AmazonBasics Microwave）和 Amazon 智慧插座（Amazon Smart Plug）。 新版 Echo Dot 使用新的表面材質並加強音響性能，播放音樂時的音量增加且音色更豐富。新版 Echo Plus 同樣採用新的表面材質和強化音響，內建 Zigbee 智慧家庭能連結相容的智慧家居裝置，不需額外的應用程式和線材。Echo Plus 還配備了溫度感應器，可以在到達特定溫度時開啟連接的裝置，像是太熱的時候自動打開電風扇。 Echo Auto 則是將智慧音箱搬到車上，可以利用汽車的電源直接充電，並使用藍牙或 3.5mm 插孔連結車上系統。Echo Auto 可運用手機 Alexa 應用程式取得 Alexa 語音助理，就可聲控操作播放音樂和導航等功能。Echo Dot 定價為 49.99 美元，Echo Auto 定價為 24.99 美元，Echo Show 定價為 229.99 美元，Echo Plus 定價則為 149.99 美元。Echo Dot 和 Echo Plus 即日起開放預購，預計在 10 月開始出貨，Echo Auto 則需要獲得邀請才能購買。 ▲ Echo Dot。 ▲ Echo Plus。  ▲ Echo Auto。 Echo Wall Clock 可相容 Echo 智慧音箱，並能以語音操控同時進行多個計時，以及提供時間提醒。Echo 掛鐘會以藍牙連結到 Echo 智慧音箱，自動同步到正確的時間。Echo Input 可以利用藍牙或 3.5mm 插孔連結到現有的音響，將智慧語音助理 Alexa 加入音響的功能，讓音響搖身一變成為智慧音響。 Echo Sub 是第一款無線 Echo 超低音音響，適合注重音樂體驗的用戶。連結 Echo 裝置之後，只要讓 Alexa 開啟低音，就能享受更好的低音效果。Echo Link 和 Echo Link Amp 則適合擁有自己的音響裝置，但還是想要用 Alexa 來聆聽音樂的用戶。Echo Link 可連結到高級音響，使用 Alexa 來播放音樂的同時還是能保持高品質的音質。Echo Link Amp 則是 Echo Link 的進階版，額外內建 60W 功率的雙聲道放大器。 Echo Wall Clock 定價為 29.99 美元，Echo Input 定價為 34.99 美元，Echo Sub 定價為 129.99 美元，Echo Link 定價為 199.99 美元，Echo Link Amp 定價則為 299.99 美元。Echo Wall Clock、Echo Input 和 Echo Link 會在 2018 年就上市，Echo Link Amp 則要等到 2019 年才會上市。Echo Sub 即日起開放預購，預計 10 月開始出貨。 ▲ Echo Input。  ▲ Echo Sub。  ▲ Echo Link。  ▲ Echo Link Amp。 Amazon 智慧微波爐外型看起來就像一般微波爐，但可以連結 Echo 裝置，讓烹調更為便利。只要按下「Ask Alexa」的按鈕，就能以語音指令讓 Amazon 智慧微波爐進行特定時間長度和特定熱度的加溫。Amazon 智慧插座連結到電燈、電風扇和咖啡機等電器，就能讓讓用戶以 Alexa 聲控開關這些電器。只要有 Echo 裝置或是具有安裝 Alexa 應用程式的任何裝置就能操控，還能設定固定時程開關電器。
